# Code Climate luacheck Engine

CodeClimate Engine that runs [luacheck](https://github.com/mpeterv/luacheck/) on Lua files.
Uses [linguist](https://github.com/github/linguist) to detect Lua files.

## Testing

```shell
make build test
```
